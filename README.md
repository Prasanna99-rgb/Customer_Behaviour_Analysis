## 📌 Project Overview

Customer Shopping Behavior Analysis is a data analytics project that explores customer purchasing patterns, demographics, subscription behavior, shipping preferences, and revenue trends. The project leverages Python, PostgreSQL, and data visualization techniques to uncover actionable business insights that can improve customer retention and increase profitability.

## 📈 Key Insights

### 👩 Revenue by Gender
- Female customers generated slightly higher revenue compared to male customers.

### 🎯 High-Value Discount Users
- Some customers consistently spend above average while utilizing discounts.
- These customers represent a valuable target audience for premium promotions.

### ⭐ Top-Rated Products
- Blouse
- Dress
- Shirt

### 🚚 Shipping Preferences
- Customers using Express Shipping spend more per transaction than Standard Shipping users.
- Faster delivery options positively correlate with purchase amount.

### 💳 Subscription Impact
- Subscribers spend significantly more than non-subscribers.
- Subscription customers contribute a large portion of total revenue.
- Higher repeat purchase rates observed among subscribers.

### 👥 Customer Segmentation

| Segment | Description |
|----------|------------|
| New Customers | First-time buyers |
| Returning Customers | Regular shoppers |
| Loyal Customers | High-value repeat customers |
